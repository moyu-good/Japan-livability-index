# 重みプリセット設計根拠

各プリセットの重み設定とその根拠を記録します。

## 次元定義

| 次元 | 構成指標 |
|------|---------|
| 経済力 | 県民所得_一人当たり、完全失業率、有効求人倍率 |
| 生活利便性 | 持ち家比率、住宅延べ面積、家賃_1畳当たり、収入対家賃比 |
| 環境快適度 | 年間日照時間、年間降水量（逆転なし） |
| 社会安全度 | 犯罪発生率、交通事故発生率（ともに逆転指標） |
| 将来性 | 人口増減率、転入超過率、高齢化率（逆転）|

## プリセット一覧

### デフォルト（等倍寄り）
| 次元 | 重み |
|------|------|
| 経済力 | 25% |
| 生活利便性 | 30% |
| 環境快適度 | 20% |
| 社会安全度 | 10% |
| 将来性 | 15% |

**根拠**: 特定ライフスタイルに偏らない汎用的なバランス。小紅書記事の「デフォルトランキング」として使用し、東京が1位にならない結果が得られるか要検証。

### 家庭向け（👨‍👩‍👧）
| 次元 | 重み |
|------|------|
| 経済力 | 15% |
| 生活利便性 | 35% |
| 環境快適度 | 20% |
| 社会安全度 | 20% |
| 将来性 | 10% |

**根拠**: 子育て世帯は居住コストと治安を最重視。経済力の重みを下げ、生活利便性・社会安全度を高める。

### キャリア（💼）
| 次元 | 重み |
|------|------|
| 経済力 | 40% |
| 生活利便性 | 30% |
| 環境快適度 | 10% |
| 社会安全度 | 10% |
| 将来性 | 10% |

**根拠**: 単身・共働きで収入水準と求人市場を最優先。このプリセットでは東京・神奈川・愛知が上位に来やすい。

### セカンドライフ（🌿）
| 次元 | 重み |
|------|------|
| 経済力 | 10% |

**根拠**: 退職後の移住検討者は気候・自然環境を最重視。山陰・四国・九州が上位に来やすい。
