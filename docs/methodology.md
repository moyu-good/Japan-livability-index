# 方法論ドキュメント

## 概要

本プロジェクトは、日本47都道府県の「生活しやすさ（Livability）」を多次元データで定量的に評価するものです。

## 評価フレームワーク

### 5大次元・合計20指標

| 次元 | 重みづけ | 説明 |
|------|---------|------|
| 経済力 | 25% | 収入・雇用・住居コスト |
| 生活利便性 | 30% | 医療・通勤・商業 |
| 環境快適度 | 20% | 天気・自然環境・災害 |
| 社会安全度 | 10% | 治安・交通安全 |
| 将来性 | 15% | 人口動態・出生率・財政 |

### スコアリング手法

1. **正規化**: 各指標をMin-Maxスケーリングで0〜100に変換
2. **方向性調整**: 「低いほど良い」指標（犯罪率・通勤時間等）は100から引いて反転
3. **加重合計**: 事前定義した重みで次元内・次元間の加重平均
4. **総合ランキング**: 全指標の加重合計を最終スコアとし順位付け

### 透明性の確保

- 全データは政府公開統計（e-Stat等）を一次ソースとして使用
- 重みづけの根拠は`docs/weight_rationale.md`に記述
- 再現性のためコードとデータはオープンソース公開

## 注意事項

- 本ランキングはデータに基づく客観的評価であり、居住地選択の唯一の判断基準にはなりません
- 都市内の地域差は都道府県単位のデータでは捉えられません
- データの収集時期によって結果は変動します
